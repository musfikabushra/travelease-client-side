## Features
- Users can **explore and book vehicles** with detailed information.  
- Authenticated users can **add, update, and delete their own vehicle listings**.  
- **Responsive UI** design suitable for mobile, tablet, and desktop.  
- Users can toggle between **dark and light** themes.
- On the **All Vehicles page**, vehicles are displayed with **filtering based on price**
- **Google social login** is integrated with **Firebase Authentication.**
- **Toast notifications** have been added to display **success/error messages.**
- Display of **recent 6 vehicles** dynamically on the homepage.

--- 

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** Firebase Authentication  
- **API Calls:** Axios
